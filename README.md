# UE5_FishingMaster_Game

A stylized realism fishing game developed in Unreal Engine 5.4, featuring dynamic ocean simulation, procedural fish swarms, and a risk-reward harvesting system.

## Setup Instructions

- **To play the game:** Extract the downloaded archive and double-click the `.exe` file to run.
- **To view the source code:** Open the `.uproject` file using Unreal Engine 5.4.

## Dependencies

- **Engine:** Unreal Engine 5.4
- **Note:** No additional third-party plugins or external libraries are required to run this project.
